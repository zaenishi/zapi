🔑 Authentication

All API calls require authentication using an API key:


/* Example using whatsapp baileys */
try {
const { data } = await axios.get(`https://zshiapi.vercel.app/videydl?apikey=xxx&url=xxx`)
if (!data) return m.reply('Data not found')
if (data.status == 401) return m.reply('Invalid Apikey')
return m.reply(data)
} catch (e) {
console.log(e)
}
Important: Keep your API key secure and never expose it in client-side code.
